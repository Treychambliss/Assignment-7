## How to Run
1. Install python on your system, as well as PySide6 (pip install PySide6)
2. Download the convert.py and house.png files and place them in the same folder.
3. Run the convert.py file using your IDE or terminal.
4. Enter your proper conversions and click convert. The conversion will be displayed.
5. Click clear to reset the program to start a new conversion or click exit to exit the program.
